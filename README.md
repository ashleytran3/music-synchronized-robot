# music-synchronized-robot

Robot: Baxter + Robotiq85

Dancing Robot (Music Synchronization): Implement beat detection for an
audio track and synchronize robot joint or end-effector motion to the detected
tempo. The system must: (1) extract tempo automatically, (2) generate
time-aligned motion patterns, and (3) demonstrate synchronization accuracy in
simulation.

## Dependencies

1. Librosa (for mac, pip install librosa)

## Setup
1. Dowload an mp3 version of "Uptown Funk" as save it in assets: /assets/uptown_funk.mp3
